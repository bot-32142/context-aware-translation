# UX Redesign Program

This directory is the working spec for the desktop app UX redesign.

Structure:
- `phase0/`: locked product architecture, terminology, and canonical journeys
- `program.md`: sequencing, ownership, dependencies, and parallelism
- `phase1_setup.md`: app setup, project setup, and provider-first onboarding
- `phase2_work.md`: Work home and ordered document pipeline
- `phase3_queue.md`: Queue drawer, blockers, and background-action UX
- `phase4_document_workspace.md`: document-scoped OCR, Terms, translation,
  image, and export surfaces
- `phase5_terms.md`: shared Terms surface and table-first glossary management
- `phase6_advanced_controls.md`: advanced controls layered onto the same shell

Read order:
1. `phase0/architecture.md`
2. `phase0/terminology.md`
3. `phase0/journeys.md`
4. `program.md`
5. phase-specific specs

This package is intentionally UX-first. It does not define implementation
schemas, migrations, or code structure.
